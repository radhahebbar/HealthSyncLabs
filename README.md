# HealthSync Labs

## AI-powered Clinical Documentation & Patient Data Extraction

### Project Overview

HealthSync Labs is an AI-powered healthcare automation solution that extracts structured patient information from unstructured clinical documents.

The solution combines:

- n8n
- Google Document AI
- Gemini AI
- OCR
- Dashboard Reporting

---

## Features

- Patient Data Extraction
- Clinical Notes Processing
- Invoice Extraction
- AI-generated Summaries
- Automated Notifications

---

## Technology Stack

- n8n
- Google Document AI
- Gemini AI
- Python / FastAPI
- Docker
- GitHub
